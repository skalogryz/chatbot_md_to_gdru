# chatbot_md_to_gdru
конвертатор markdown разметки чатботов, в гд.ру разметку

на 2026 год появилось много бесплатных чатботов.
Люди с удовольствиям постят их выхлопы, хотя markdown разметка не поддерживается (форумами)

Неоформленный текст раздражает; гигантские объёмы нейробреда читать без форматирования нет никакого удовольствия.
По-этому их нужно хотя бы отформатировать в дружелёбный для форума вид.
